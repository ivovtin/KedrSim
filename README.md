# Пакет полного моделирования детектора КЕДР

Для получения кода необходимо выполнить следующую команду:
```
git clone https://github.com/ivovtin/KedrSim
```

Для сборки нужно выполнить в текущей директории:
```
make
```

Запуск моделирования:
```
~/bin/ks < mc.cards > /dev/null
```

генератор космических мюонов (автор - Балдин Е.М.)  <br />
GENE 53 0.01 5. 0. 85. -32.5 32.5 12.5              <br />

Запуск моделирования на вычислительном кластере:
```
qsub batch_test_atcmc.sh
```

Запуск в KDisplay просмотра моделирования:                    <br />
```
KDisplay < simout/sim000001.dat -r -R19697
```

Запуск в KDisplay просмотра экспериментальных заходов        <br />
```
bzcat /space/data2/KEDR_RUNS/runs/daq019697.nat.bz2 | KDisplay -r -d3
```
